# ⚠️ Известные проблемы для Context7 Researcher Skill

## Проблема: Library not found
**Симптом:** Context7 возвращает "No matching libraries found"

**Решение:**
1. Попробуйте другие ключевые слова
2. Используйте более общий термин
3. Проверьте точное название библиотеки

## Проблема: Rate limiting
**Симптом:** "Rate limit exceeded" от Context7

**Решение:**
1. Подождите несколько минут
2. Используйте API ключ для увеличения лимитов
3. Кэшируйте результаты запросов

## Проблема: Outdated documentation
**Симптом:** Примеры кода не работают

**Решение:**
1. Укажите конкретную версию в libraryId
2. Используйте `/org/project/version` формат
3. Проверьте дату последнего обновления
