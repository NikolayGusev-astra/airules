# ⚠️ Известные проблемы для RAG Expert Skill

## Проблема: Poor retrieval quality
**Симптом:** Поиск возвращает нерелевантные документы

**Решение:**
1. Проверьте качество эмбеддингов
2. Улучшите размер чанков (chunk size)
3. Попробуйте другие embedding модели

## Проблема: Hallucination
**Симптом:** AI генерирует неверную информацию

**Решение:**
1. Добавьте больше релевантного контекста
2. Используйте grounding (ссылки на источники)
3. Уменьшите temperature генерации

## Проблема: Slow query processing
**Симптом:** Время ответа > 10 секунд

**Решение:**
1. Создайте векторные индексы
2. Используйте кэширование
3. Примените approximate search (HNSW, IVF)
